# なにわ男子紹介サイト

アイドルグループ「なにわ男子」を紹介するファンサイトとして制作したWebサイトです。

## 概要
メンバー紹介やCD・DVD・アルバム情報などを掲載しています。  
メンバーカラーをテーマにした「ラッキーカラー占い」機能も実装しました。

## 使用技術
- HTML
- CSS
- JavaScript

## 特徴
- メンバー紹介ページ
- CD / DVD / アルバム紹介
- メンバーカラーを使ったラッキーカラー占い機能
